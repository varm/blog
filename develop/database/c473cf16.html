DBeaver 是一款基于 Java 开发的跨平台的强大数据库管理工具。

跨平台

  • Windows
  • Mac OS X
  • Linux

版本

  • 社区版（开源，免费）

    通过 JDBC 连接到数据库，可以支持几乎所有的数据库产品，包括：MySQL、PostgreSQL、MariaDB、SQLite、Oracle、Db2、SQL Server、Sybase、MS Access、Teradata、Firebird、Derby 等等。

破解

  1. 下载安装 DBeaver

    可以直接下载 zip 压缩包，解压后比如放在：D:\Program\dbeaver。

  2. 下载破解文件

    • DBeaver破解-天翼云
    • DBeaver破解-蓝奏云

    DBeaver

    https://cloud.189.cn/t/R7BRzqnYVJNb

    (访问码：jgd1)

    DBeaver

    https://varm.lanzoui.com/iqwWhpwmcwd

    (访问码：9527)

    下载完成后将 jar 文件也放置在 DBeaver 安装目录：D:\Program\dbeaver。

  3. 打开DBeaver 配置文件：dbeaver.ini，当前下载的 21.2.0 版本内容是这样的：

    -startup
plugins/org.eclipse.equinox.launcher_1.6.100.v20201223-0822.jar
--launcher.library
plugins/org.eclipse.equinox.launcher.win32.win32.x86_64_1.2.100.v20210209-1541
-vmargs
-XX:+IgnoreUnrecognizedVMOptions
--add-modules=ALL-SYSTEM
-Dosgi.requiredJavaVersion=11
-Xms128m
-Xmx2048m

    然后在最后一行添加：

    -javaagent:D:\Program\dbeaver\dbeaver-agent.jar

  4. 点击 dbeaver.exe 启动 DBeaver

    出错了，因为本地默认安装的 Java 版本是 JDK 17，所以需要修改一下配置文件。

  5. 打开 dbeaver.ini，指定 jdk 17 的本地路径，最终修改文件内容如下：

    -startup
plugins/org.eclipse.equinox.launcher_1.6.100.v20201223-0822.jar
--launcher.library
plugins/org.eclipse.equinox.launcher.win32.win32.x86_64_1.2.100.v20210209-1541
-vm
D:\Program Files\Java\jdk-17\bin\javaw.exe
-vmargs
-XX:+IgnoreUnrecognizedVMOptions
--add-modules=ALL-SYSTEM
-Dosgi.requiredJavaVersion=11
-Xms128m
-Xmx2048m
-javaagent:D:\Program\dbeaver\dbeaver-agent.jar

    -vm 选项必须在 --launcher.library 之后和 -vmargs 之前

  6. 重新启动 DBeaver，大功告成！
