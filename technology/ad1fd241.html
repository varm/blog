Windows 预览版用户升级到 Windows 11

technology

发布于：2021年6月30日
次浏览

环境配置

  1. 本人 VMware Workstation 中已经安装了一个 Windows 10 Pro，系统版本已经更到 21H1，开启了【预览体验计划】中的 Release preview channel。

  2. 因为 Windows 11 对于 CPU 有一个最低配置要求：

    需要 1 GHz 或更快的支持 64 位的处理器（双核或多核）或系统单芯片 (SoC)

所以需要在 Vmware 中将处理器核心至少设置为 2，否则无法升级成功

升级步骤

  1. 打开【设置】->【Windows 预览体验计划】，确认系统已经加入 “Insider 计划”并选择 Release Preview

  2. 点击 Windows+R，输入：“regedit”，打开注册表编辑器

  3. 定位到：HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsSelfHost\UI\Selection

    1. 将 UIBranch 中的值更改为 Dev
    2. 将 ContentType 中的值更改为 Mainline
    3. 将 Ring 中的文本更改为 External

  4. 定位到：HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsSelfHost\Applicability

    1. 将 BranchName 中的值更改为 Dev
    2. 将 ContentType 中的值更改为 Mainline
    3. 将 Ring 中的值更改为 External

  5. 关闭注册表编辑器，重启电脑

  6. 打开【设置】->【Windows 预览体验计划】，查看是否从之前的“Release Preview”频道变更到“Dev渠道”

  7. 点击【Windows 更新】，正常情况下可以马上收到 Windows 11 更新

开始升级
